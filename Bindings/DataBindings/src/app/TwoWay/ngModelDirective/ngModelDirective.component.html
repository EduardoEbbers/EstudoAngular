Estrutura Básica Diretiva NgModel Binding

Usa-se para gerenciar a propriedade class.

Mais flexíveis em relação ao retorno da expressão, que pode ser:

  • String

    São adicionados classes especificadas através de um string.

    Obs.: Múltiplas classes são separadas por espaços.

  • Array (Lista)

    Cada objeto do array representa uma classe que vai ser adicionado no elemento.

  • Object (Map)

    Cada propriedade do objeto representa uma class ou mais.

    Mais usado.

    Obs.: Múltiplas classes em uma propriedade do objeto são separadas por espaços.

    Obs.: A classe será adicionada caso o valor da propriedade for truthy, visto aqui .

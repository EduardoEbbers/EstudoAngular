Diretiva ngTemplateOutlet Bindings


Estrutura Básica

Usa-se para repetir blocos de conteúdos em um local especificado

Quando se quer gerar o mesmo conteúdo em diferentes lugares

Primeiro deve-se definir o template que contem o conteúdo que vai ser repetido

Que é um elemento ng-template

Que possui variavel de referência (reference variable), que pode ser usado em outras partes do HTML

Segundo deve-se usar o elemento ng-template fazendo um binding com a diretiva ngTemplateOutlet

Assim a expressão deve ser o mesmo nome da variável de referência que foi atribuída no elemento ng-template que se dá a repetição.

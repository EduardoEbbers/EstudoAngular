One Way

Diretiva ngTemplateOutlet Bindings


Estrutura Básica

Usa-se colchetes aplicados ao redor de uma propriedade padrão do elemento host que gerenciam o estado do elemento

Assim a expressão é avaliada e setada para o alvo ou propriedade do elemento host modificando o estado ou conteudo

Para saber mais sobre elemento host e suas propriedades e metodos consultar o website do monzila

developer.mozilla.org/en-US/docs/Web/API/HTMLElement

  • Lista de Exemplos

    • Exemplo 1: Propriedade href da tag anchor
      Seta href

      A propriedade "href" do elemento host anchor (ou tag a), serve para setar um endereço URL válido após avaliação da expressão do binding

      A expressão do binding"getHref()" é um método do componente do template HTML.

      No esquema acima, Angular seta o binding na propriedade [href] após avaliação do método getHref()

      O retorno do método getHref() no componente é uma string, que avalia se a quantidade de produtos é igual a 5

      caso sim, o endereço URL será "/bindings/oneWay/standardProperty)", caso não, o endereço URL será a página inicial.

      Aqui temos a tag anchor (a) que possui um binding na propriedade id e title do elemento host.

      A expressão "getId()" e "getTitle()" são métodos do componente do template HTML.

      No esquema acima, Angular seta o binding na propriedade [href] após avaliação do método getHref()

      O retorno do método getHref() do componente é um string, que avalia se a quantidade de produtos é igual a 5, caso sim (o retorno será /bindings/oneWay/standardProperty), caso não (o retorno será retornar /)

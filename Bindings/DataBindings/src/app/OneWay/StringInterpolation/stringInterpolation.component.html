Estrutura Básica String Interpolation Binding

Usa-se dupla chaves que envolve o resultado da expressão.

Assim a expressão é avaliada e seta o conteúdo do elemento host.

Obs.: String Interpolation deve-se usar para simples fragmento de código.

Obs.: O retorno da expressão é um string e equivalente a fazer um data binding na propriedade textContent de certos elementos host.

Nota: Para saber mais sobre a propriedade textContent consultar o website1 e website2 .

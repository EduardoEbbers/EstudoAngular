Estrutura Básica String Interpolation Bindings

Usa-se dupla chaves que envolve o resultado da expressão.

Assim a expressão é avaliada e setada como string no conteúdo do elemento host.

Obs 1: O retorno da expressão é um string equivalente a usar a propriedade "textContent"

Style Binding


Estrutura Básica

Propriedade do elemento host

Existe 3 formas de manipulação:

  • Standard Property Binding

    Estrutura basica

    [style.mystyle]

    Seta um único valor de uma única propriedade style

    Obs 1: Deve-se incluir as unidades requeridas para o style

  • Style Binding Special

    Estrutura basica

    [style.myStyle.units]

    Seta um único valor de uma única propriedade style

    Obs 1: Não deve incluir as unidades requeridas

  • diretiva ngStyle

    Estrutura basica

    [ngStyle]

    Explicado em diretiva ngStyle

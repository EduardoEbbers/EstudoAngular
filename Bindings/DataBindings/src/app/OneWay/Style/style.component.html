Style Binding


Propriedade style do elemento host

Existe 3 formas de manipulação desta propriedade:

  • Standard Property Binding

    Estrutura basica


    Seta um único valor de uma única propriedade style

    Obs 1: Deve-se incluir as unidades requeridas para o style no componente.

  • Style Binding Special

    Estrutura basica


    Seta um único valor de uma única propriedade style

    Obs 1: Não deve incluir as unidades requeridas

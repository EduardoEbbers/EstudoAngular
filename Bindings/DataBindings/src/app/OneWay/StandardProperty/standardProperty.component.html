Standard Property Bindings


Estrutura Básica

Usa-se colchetes aplicados em propriedades padrão do elemento host que gerenciam o estado do elemento

Assim a expressão é avaliada e setada para o alvo ou propriedade do elemento host modificando o estado ou conteudo


Colocar o exemplo aqui ? Obs 1: Caso o objeto de dados possua um valor nulo, o operador "?", na expressão acima, é uma condicional que, caso o retorno seja nulo, tenha-se um resultado descritivo no input como "None"

ex, propriedade "value" do elemento host input, serve para setar conteudo do input após avaliação da expressão do binding

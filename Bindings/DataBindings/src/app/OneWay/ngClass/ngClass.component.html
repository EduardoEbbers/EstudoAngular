Estrutura Básica
Estrutura Básica Diretiva NgClass Binding

Usa-se para gerenciar a propriedade class.

Mais flexíveis em relação ao retorno da expressão, que pode ser:

  • String

    São adicionados classes especificadas através de um string.

    Obs.: Múltiplas classes são separadas por espaços.

  • Array (Lista)

    Cada objeto do array representa uma classe que vai ser adicionado no elemento.

  • Object (Map)

    Cada propriedade do objeto representa uma class ou mais.

    Mais usado.

    Obs.: Múltiplas classes em uma propriedade do objeto são separadas por espaços.

    Obs.: A classe será adicionada caso o valor da propriedade for truthy, visto aqui .

  • Lista de Exemplos
    • Exemplo 1: Usando método com retorno String
      O primeiro produto é {{ repository.getProduct(1).name }}
      O segundo produto é {{ repository.getProduct(2).name }}
      O terceiro produto é {{ repository.getProduct(3).name }}
    • Exemplo 2: Usando método com retorno Array
      O primeiro produto {{ repository.getProduct(1).name }} tem o preço {{ repository.getProduct(1).price }}
      O segundo produto {{ repository.getProduct(2).name }} tem o preço {{ repository.getProduct(2).price }}
      O terceiro produto {{ repository.getProduct(3).name }} tem o preço {{ repository.getProduct(3).price }}
    • Exemplo 3: Usando método com retorno Object
      O primeiro produto {{ repository.getProduct(1).name }} tem o preço {{ repository.getProduct(1).price }}
      O segundo produto {{ repository.getProduct(2).name }} tem o preço {{ repository.getProduct(2).price }}
      O terceiro produto {{ repository.getProduct(3).name }} tem o preço {{ repository.getProduct(3).price }}

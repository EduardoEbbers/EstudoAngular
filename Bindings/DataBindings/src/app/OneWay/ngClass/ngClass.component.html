Diretiva ngClass Bindings


Estrutura Básica

Usa-se para gerenciar propriedade class

Mais flexiveis em relação ao retorno da expressão que pode ser:

  • String

    • São adicionados classes especificadas por um string

    • Obs. 1: Múltiplas classes são separados por espaços

  • Array

    • Cada objeto do array representa uma classe que vai ser adicionado no elemento

  • Object (map)

    • Cada propriedade do objeto representa uma class ou mais, separados por espaços

    • Mais útil

    • Obs. 2: A classe será adicionada caso o valor da propriedade for truthy


  • Lista de Exemplos

    • Exemplo 1: Diretiva ngClass
      O primeiro produto é {{ repository.getProduct(1).name }}
      O segundo produto é {{ repository.getProduct(2).name }}
      O terceiro produto é {{ repository.getProduct(3).name }}

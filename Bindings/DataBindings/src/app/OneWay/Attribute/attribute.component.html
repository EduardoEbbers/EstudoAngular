Estrutura Básica Attribute Binding

Deve-se usar quando não se tem o atributo do elemento HTML na propriedade equivalente do API DOM, como colspan.

Usa-se o prefixo attr. com o nome do atributo dentro de colchetes para setar atributos do elemento host.

Nota: Existem diferenças entre colspan e colSpan, para saber mais consultar o website .

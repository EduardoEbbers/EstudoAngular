Class Binding


Estrutura Básica

Propriedade do elemento host propriedade class do elemento host

Existe 3 formas de manipulação:

  • Standard Property Binding

    Estrutura basica

    [class]

    Seta todas as classes de um elemento após a avaliação da expressão, em uma única etapa, através de uma propriedade ou método do componente

    Obs 1: A expressão deve ter um retorno string

    Obs 2: Os nomes das classes devem ser separadas por espaços

  • Class Binding Special

    Estrutura basica

    [class.myClass]

    Seta uma única classe existente através de um controle mais refinado usando uma expressão

    Obs 1: A expressão deve ter um retorno baseado truthy, ou seja, nao pode conter os seguintes resultados de falsidade (falsy):

    • Booleano false
    • Valor númerico 0 (zero)
    • String vazio ("")
    • Null
    • Undefined
    • Valor númerico especial NaN

  • diretiva ngClass

    Estrutura basica

    [ngClass]

    Explicado em diretiva ngClass

Class Binding

Propriedade class do elemento host.

Existe 3 formas de manipulação desta propriedade:

  • Standard Property Binding
    Estrutura Básica
    Estrutura Básica Class Binding

    Seta, em uma única etapa, todas as classes de um elemento HTML após a avaliação da expressão através de uma propriedade ou método do componente.

    Obs.: A expressão deve ter um retorno string.

    Obs.: Caso houver mais de uma classe no retorno do componente, deve-se separar por espaço.

  • Special Class Binding
    Estrutura Básica
    Estrutura Básica Class Binding

    Seta uma única classe através de um controle mais refinado usando uma expressão.

    Usa-se a propriedade class. com o nome da classe dentro de colchetes para modificar o elemento host.

    Obs.: A expressão deve ter um retorno baseado em truthy, visto aqui .

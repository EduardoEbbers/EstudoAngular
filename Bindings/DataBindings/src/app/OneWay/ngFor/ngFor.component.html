Diretiva ngFor Bindings


Estrutura Básica

Usa-se quando se quer repetir a secção de conteúdos para cada objeto de um array.

Obs. 1: Equivale com loop foreach do JavaScript

Obs. 1: O asterisco (*) indica que a diretiva é um micro-template, ver aqui .

A expressão se divide em duas partes:

  • Variável de Template (Template Variable)

    Usa-se a keyword let + nome_variável_template.

    A variável de template recebe os dados recebidos do data source para serem usados dentro do template, representando cada objeto do array.

  • Fonte de Dados (Data Source)

    Fornece a fonte de dados que será enumerado, sendo o array de objetos.

São eles:

  • Index
  • Count
  • Odd
  • Even
  • First
  • Last
  • GetKey

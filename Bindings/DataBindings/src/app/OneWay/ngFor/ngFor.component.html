Diretiva ngFor Bindings


Estrutura Básica

Usa-se quando se quer repetir a secção de conteúdos para cada objeto de um array

Obs. 1: Parecido com loop foreach

A expressão se divide em duas partes:

  • Variável de template

    • Usa-se keyword let

    • é como os dados são passados dentro do template, usado como por exemplo item.name

    • let item

  • Fonte de Dados

    • Fornece o data source que sera enumerado

    • getProdcuts()

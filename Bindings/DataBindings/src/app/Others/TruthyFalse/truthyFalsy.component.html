Estrutura Básica Truthy/Falsy

A expressão sempre deverá ter um retorno baseado em truthy (de veracidade).

Para isso, a expressão não pode conter os seguintes resultados falsy (de falsidade):

  • Booleano, false
  • Valor númerico, 0 (zero)
  • String vazio, ""
  • Null
  • Undefined
  • Valor númerico especial, NaN

Obs. 1: Truthy/Falsy x True/False

Um String com valor "false" é Truthy por ser preenchido e possui valor diferente que vazio "", ele não é um boolean false nem Falsy, será Truthy po Não confundir Truthy/Falsy com os valores booleanos true/false

Sendo assim, ele não é um boolean false, porque é String, e nem Falsy, porque não é Sring vazio ""
